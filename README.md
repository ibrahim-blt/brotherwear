BROTHERWEAR PROJE BİLGİLERİ

--PROJE AÇIKLAMASI--

BrotherWear, HTML, CSS, JavaScript, PHP ve SQL kullanılarak hazırlanmış örnek bir online kıyafet satış sitesidir.

Projede kullanıcı kayıt-giriş işlemleri, ürün listeleme, ürün detay görüntüleme, beden/numara seçimi, sepete ürün ekleme, kupon uygulama, ödeme işlemi, profil sayfası ve sipariş takibi gibi temel e-ticaret özellikleri bulunmaktadır.

--KURULUM NOTLARI--

İndirdiğiniz BrotherWear proje klasörünü, dosyalardan C:\xampp\htdocs dizininin içine kopyalayın.(!önemli)

Projeyi çalıştırmadan önce XAMPP üzerinden Apache ve MySQL servisleri başlatılmalıdır.

Proje dosyaları C:\xampp\htdocs\brotherwear klasörü içerisine yerleştirilmelidir.

Veritabanı dosyası phpMyAdmin üzerinden içe aktarılmalıdır.

Site dosya olarak açılmamalıdır. Proje tarayıcı üzerinden localhost bağlantısı ile çalıştırılmalıdır.

Örnek çalıştırma bağlantısı: http://localhost/brotherwear/anasayfa.html
